Experiment 7 - Reverse Mad Libs

Description

Normally in the game Mad Libs, one player asks another player for verbs, nouns, adjectives and other kinds of words to fill in a ridiculous story revealed at the end. In this project however, as you type the story, the program will every now and then guess what type of word (noun, verb, adjective, etc) and choose a random one to add to your story. The word data at the bottom shows the suredness of each type of word.

Up/Down arrow - scroll up / down

ENTER - force fill the next word

Technical

Using brain.js I created and trained a network that is inputed a string in the form of an array of ASCII numbers with the appropriate output being what kind of word should come next. For example input: "I love to" output: Verb. After training, the program will try to find a suitable word based on the end of the text whenever the user presses SPACE. The brain thinks of which type of word is most suitable and will grab a random word of that type from a dictionary of arrays and print that word.
