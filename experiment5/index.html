Experiment 4 - Scatterbrain

Description

Watch the inner macinations of this figures mind.

R = Reset grid

Q = Flip through fill modes

E = Flip through stroke modes

Technical

Based on the size of the canvas, a 3D array is created representing different coordinates. Then an array representing a path attempts to walk as far as possible without running into itself by checking all possible neighbors. If there are no options, then the path ends. Each frame, this path is reconstructed using cubes and lines. As this path is being drawn out the path the starting value of the path is also being shifted out to make the path appear to be shrinking.
